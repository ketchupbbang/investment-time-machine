# 🎢 장기 투자자 시뮬레이터

과거 시점으로 돌아가 적립식 투자의 등락을 체험해볼 수 있는 웹 시뮬레이터입니다.

## ✨ 주요 기능

### 📊 투자 시뮬레이션
- **적립식 투자 체험**: 초기 거치금 + 정기 입금으로 장기 투자 시뮬레이션
- **입금 주기 선택**: 매월(월초) 또는 매주(월요일) 입금
- **년도별 입금액 설정**: 각 년도마다 다른 입금액 설정 가능
- **자동 증액 옵션**: 매년 n% 자동 증액 기능

### ⏱️ 시뮬레이션 컨트롤
- **속도 조절**: ms 단위로 직접 입력하여 속도 조절
- **날짜 점프**: 특정 날짜로 바로 이동
- **1년 점프**: 버튼 한 번으로 1년 빠르게 스킵
- **키보드 컨트롤**: 
  - `Space`: 일시정지/재개
  - `←` `→`: 하루씩 이동 (일시정지 상태에서)

### 📜 투자 기록
- **실시간 로그**: 배당금 재투자, 액면분할, 정기 입금 기록
- **역사적 사건 알림**: 주요 경제 이벤트 하이라이트 표시
  - 2008 리먼브라더스 파산
  - 2020 코로나 폭락/저점
  - 2022 금리인상 하락장 등

### 📈 시각화
- **실시간 차트**: 평가 금액 vs 투자 원금 비교 그래프
- **대시보드**: 총 자산, 수익률, 보유 수량, 투입 원금 표시

## 🚀 실행 방법

### 방법 1: Live Server (VS Code)
1. VS Code에서 프로젝트 폴더 열기
2. `index.html` 우클릭 → **Open with Live Server**

### 방법 2: Python 내장 서버
```bash
cd investment-time-machine
python -m http.server 8000
```
브라우저에서 `http://localhost:8000` 접속

### 방법 3: GitHub Pages
이 저장소를 GitHub Pages로 호스팅하면 바로 사용 가능합니다.

> ⚠️ **주의**: `file://` 프로토콜로 직접 열면 CORS 오류가 발생합니다. 반드시 웹 서버를 통해 실행하세요.

## 📸 스크린샷

### 설정 화면
- 종목 선택 및 데이터 기간 확인
- 시작일/종료일 설정

## 📝 사용 예시

1. **종목 선택**: VOO, QQQ 등 원하는 종목 선택
2. **기간 설정**: 시작일과 종료일 입력 (예: 2010-01-01 ~ 2023-12-31)
3. **투자 설정**: 
   - 초기 거치금: $10,000
   - 정기 입금액: $1,000/월
   - 년도별 증액: 매년 5% 증액 (선택)
4. **체험 시작하기** 클릭
5. 과거의 등락을 체험하며 장기 투자의 중요성 확인!

## ⚠️ 주의사항

- 과거 수익률이 미래 수익률을 보장하지 않습니다.
- 실제 투자 결정은 전문가와 상담 후 신중하게 진행하세요.
